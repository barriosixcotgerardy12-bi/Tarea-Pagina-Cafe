## Descripción
Página web de la Cafetería El Aroma desarrollada con HTML y CSS.
Contiene 3 páginas: Inicio, Menú y Reservas.
 
---
 
## Páginas del Proyecto
 
| Archivo | Descripción |
|---|---|
| `Page1.html` | Página de Inicio — Bienvenida e información |
| `Page2.html` | Página de Menú — Bebidas y Repostería |
| `style.css` | Estilos de todas las páginas |

## Etiquetas HTML Utilizadas
 
| Etiqueta | Uso |
|---|---|
| `<!DOCTYPE html>` | Define el tipo de documento |
| `<html>` | Elemento raíz de la página |
| `<head>` | Contiene metadatos de la página |
| `<meta>` | Define el charset UTF-8 |
| `<title>` | Título de la página en el navegador |
| `<link>` | Enlaza el archivo style.css |
| `<body>` | Contenido visible de la página |
| `<header>` | Encabezado con título y navegación |
| `<nav>` | Barra de navegación entre páginas |
| `<a>` | Enlaces entre páginas |
| `<main>` | Contenido principal |
| `<section>` | Secciones de contenido |
| `<h1>` | Título principal del sitio |
| `<h2>` | Subtítulo de sección |
| `<h3>` | Subtítulo de artículo |
| `<h5>` | Subtítulo de presentaciones |
| `<h6>` | Subtítulo de notas |
| `<p>` | Párrafos de texto |
| `<strong>` | Texto en negrita |
| `<em>` | Texto en cursiva |
| `<hr>` | Línea horizontal separadora |
| `<ul>` | Lista desordenada |
| `<ol>` | Lista ordenada |
| `<li>` | Elemento de lista |
| `<table>` | Tabla de productos y precios |
| `<tr>` | Fila de tabla |
| `<th>` | Encabezado de columna |
| `<td>` | Celda de tabla |
| `<img>` | Imagen de producto |
| `<footer>` | Pie de página |
 
---
 
## Estilos CSS Utilizados
 
| Propiedad | Uso |
|---|---|
| `font-family` | Tipo de letra Arial |
| `background-color` | Color de fondo del body, header, section y footer |
| `color` | Color del texto en h2, h3, h5, h6, p, li y nav |
| `margin` | Espacio exterior en body y sections |
| `padding` | Espacio interior en header, section, footer, th y td |
| `text-align` | Alineación del texto en header y footer |
| `text-decoration` | Quitar subrayado en enlaces del nav |
| `font-weight` | Texto en negrita en nav y th |
| `font-size` | Tamaño del texto en nav |
| `border` | Borde en sections y tablas |
| `border-collapse` | Colapsar bordes de la tabla |
